# Antonio Joshy - Creative Technologist Portfolio

A modern, production-ready 3D portfolio website built with React, Three.js, and Tailwind CSS.

## 🚀 Features

- **3D Interactive Hero Section** - Animated 3D elements using React Three Fiber
- **Dark Glassmorphism Design** - Modern dark theme with neon blue/purple accents
- **Smooth Animations** - Powered by Motion (Framer Motion)
- **Responsive Design** - Mobile-first approach, works on all devices
- **Project Showcase** - Interactive cards with hover effects
- **Skills Section** - Animated progress bars
- **Photo Gallery** - Masonry layout with lightbox preview
- **Contact Form** - Beautiful contact section with social links
- **Loading Screen** - Animated loading experience
- **Smooth Scrolling** - Active section highlighting in navigation

## 🛠️ Tech Stack

### Frontend
- **React 18.3** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS 4** - Styling
- **Motion** (Framer Motion) - Animations
- **React Three Fiber** - 3D graphics
- **@react-three/drei** - Three.js helpers

### Additional Libraries
- **react-responsive-masonry** - Gallery layout
- **lucide-react** - Icons
- **sonner** - Toast notifications

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🎨 Design System

- **Primary Colors**: Black (#000000), Deep Gray (#1a1a1a)

- **Theme**: Dark with glassmorphism effects
- **Typography**: Clean, modern sans-serif
